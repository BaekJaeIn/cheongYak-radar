# v3 변경요청 확인 질문 — 청약시작일 캘린더 추가

**요청 요약**: 공고의 청약신청시작일(apply_start)을 Google 캘린더 템플릿 링크 방식으로 캘린더에 추가한다. 링크를 열면 기기에 로그인된 본인 Google 계정의 일정 추가 화면이 미리 채워진 채 열리며, 삼성캘린더에는 Google 계정 동기화를 통해 표시된다. 계정 연동·OAuth·서버 변경 없음.

**참고(제약)**: Google 캘린더 템플릿 링크는 알림(리마인더) 시각을 지정할 수 없습니다. 일정 저장 시 사용자의 Google 캘린더 기본 알림 설정이 적용됩니다.
